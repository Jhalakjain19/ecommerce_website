# DAP-FASHION

E_COMMERCE WEBSITE 

MADE ON PHP/MYSQL/JAVASCRIP/CSS/HTML

Install Xampp

Add Folder to C:\xampp\htdocs\

Run Apache & SQL - from Xampp Control Panel

Import SQL file(`database/fp.sql`) in phpmyadmin. And create DATABASE

Browse on Browser
